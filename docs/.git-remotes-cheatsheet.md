# Git Remotes Quick Reference

## Quick Commands

### View All Remotes
```bash
git remote -v
```

### Fetch from All Remotes
```bash
git fetch --all
```

### Fetch from Specific Remote
```bash
git fetch origin      # Primary repository
git fetch upstream    # Secondary repository
```

### Pull from Specific Remote
```bash
git pull origin main
git pull upstream main
```

### Push to Specific Remote
```bash
git push origin <branch-name>
git push upstream <branch-name>
```

### Push to Both Remotes
```bash
git push origin <branch-name> && git push upstream <branch-name>
```

### View All Branches (Local and Remote)
```bash
git branch -a
```
